## Project Overview
This project demonstrates an interactive dashboard created using Plotly to explore video game sales data. The dashboard includes interactive filters and multiple coordinated visualizations that allow users to explore sales trends by genre, publisher, and release year.

- `assignment3.ipynb` – Dashboard notebook and tutorial
